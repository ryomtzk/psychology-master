# ゼロから学ぶ心理学 🧠 — 学部心理学 総合学習サイト

研究の中で**心理学の知識が必要になった研究者**のための、学部レベル心理学を網羅的に学べるローカル学習サイトです。
**ビジュアル解説（SVG図解）**＋**間隔反復による反復演習**で、知識を着実に定着させます。

## 使い方（ローカルで動かす）

### 方法1：ダブルクリックするだけ（サーバー不要）
`index.html` をブラウザで開くだけで動きます。
（外部通信・ビルド不要。学習データはブラウザの `localStorage` に保存されます。）

### 方法2：ローカルサーバーで開く（推奨・どの環境でも確実）
```bash
cd "心理学があるフォルダ"
python3 -m http.server 8123
# ブラウザで http://localhost:8123 を開く
```

## 収録内容

- **53分野 / 189レッスン / 演習960問 / 用語1411語**
- 各分野＝「である調の網羅的解説（重要語ハイライト・コールアウト・表・SVG図解）」＋「難易度別の4択演習（解説付き）」
- うち **視覚科学（特別編）の14分野**は、視覚心理・視覚認知科学を掘り下げた特別セクション。トップナビ「👁️ 視覚科学」から専用ハブ（`#/vision`）に入れる。
- さらに **楽観性の心理学（特別編）の10分野**は、楽観性バイアス（optimism bias）の研究史を体系的にたどる特別セクション。Weinstein の非現実的楽観主義からTaylor & Brown のポジティブ・イリュージョン、計画錯誤、Sharot らの信念更新の非対称性（神経科学）、抑うつとの鏡像関係、そして現象の存在自体を問う統計的人工産物説の論争まで、「確立された知見」と「方法論的論争」を両面から扱う。トップナビ「🌈 楽観性」から専用ハブ（`#/optimism`）に入れる。
- さらに **横文字（カタカナ語）の特別編11分野**は、心理・認知科学の論文で頻出する横文字（モデレーター／アフォーダンス／プレディクティブコーディング等）を分野ごとに**頻出順**で整理した語彙セクション。演習は**意味から単語を当てる（意味→単語）**形式。トップナビ「🔤 横文字」から専用ハブ（`#/katakana`）に入れる。
- さらに特別コース **「ベイズの階段」（`bayes/`）** を同梱。中学数学の確率から条件付き確率・ベイズの定理・MCMC・PyMC実践まで、全6レベル38レッスン＋付録で学べるベイズ統計の読み物コース。トップナビ「🪜 ベイズ統計」またはホームのバナーから入れる（進捗・テーマは心理学サイトと連動）。

## 主な機能

- **📖 学習ビュー** — 目次サイドバー付き。図解・表・重要語集で視覚的に理解。
- **✏️ 反復演習** — 4択クイズ。即時に正誤＋解説を表示。選択肢は毎回シャッフル。キーボード操作対応（A〜Dで選択、Enterで次へ）。
- **🔁 間隔反復（Leitnerボックス方式）** — 間違えた問題ほど短い間隔で、覚えた問題は長い間隔で自動的に復習キューへ。ナビの「復習」に出題予定数が表示されます。
- **📝 模擬試験** — 全分野または特定分野から、10〜60問をランダム出題。
- **📊 進捗ダッシュボード** — 分野ごとの習熟度・正答率・復習待ち数を一覧表示。
- **📕 用語集** — 全1411語を横断検索。
- **👁️ 視覚科学ハブ** — トップナビ「視覚科学」から、特別編14分野を学びの順（神経基盤→中次の知覚→高次認知→理論・意識→記憶・応用）でたどれる専用ページ。
- **🌈 楽観性ハブ** — トップナビ「楽観性」から、特別編10分野を研究史の順（現象の発見→概念の整理→神経メカニズム→境界条件と論争→応用）でたどれる専用ページ。論争を扱う回は両論を併記。
- **🔤 横文字ハブ** — トップナビ「横文字」から、カタカナ語特別編11分野を「研究の数理と方法→脳・知覚・学習→言語・発達・感情→研究文化とメタ理論」の順でたどれる専用ページ。各分野は頻出順の語リスト＋意味→単語の4択演習。
- **🌙 ダーク/ライトテーマ切替**

## 学習の進め方（おすすめ）
1. 「基礎・方法論」から順に **学ぶ → 演習する** を回す。
2. 毎回まず **「復習」** で出題予定の問題を片付ける（間隔反復で長期記憶へ）。
3. 各分野がひととおり済んだら **「模擬試験（全分野）」** で実力チェック。

## ファイル構成
```
index.html              … エントリポイント（各分野スクリプトを読み込み）
assets/css/style.css    … デザイン
assets/js/app.js        … 共通エンジン（ルーター / クイズ / 間隔反復 / 進捗 / 用語集 / 模擬試験）
data/<分野>.js          … 各分野のコンテンツ（PsychApp.registerModule で自己登録）
bayes/                  … 特別コース「ベイズの階段」（独立した静的サイト。bayes/index.html がトップ）
```
新しい分野や問題を足したいときは、`data/intro.js` を雛形に `data/` へ `.js` を追加し、`index.html` の `<script>` に1行加えるだけです。

学習データのリセットは「進捗」ページ下部のボタンから行えます。
