# ゼロから学ぶ心理学 🧠 — 学部心理学 総合学習サイト

研究の中で**心理学の知識が必要になった研究者**のための、学部レベル心理学を網羅的に学べるローカル学習サイトです。
**ビジュアル解説（SVG図解）**＋**間隔反復による反復演習**で、知識を着実に定着させます。

## 使い方（ローカルで動かす）

### 方法1：ダブルクリックするだけ（サーバー不要）
`index.html` をブラウザで開くだけで動きます。
（外部通信・ビルド不要。学習データはブラウザの `localStorage` に保存されます。）

### 方法2：ローカルサーバーで開く（推奨・どの環境でも確実）
```bash
cd "心理学があるフォルダ"
python3 -m http.server 8123
# ブラウザで http://localhost:8123 を開く
```

## 主な機能

- **📖 学習ビュー** — 目次サイドバー付き。図解・表・重要語集で視覚的に理解。
- **✏️ 反復演習** — 4択クイズ。即時に正誤＋解説を表示。選択肢は毎回シャッフル。キーボード操作対応（A〜Dで選択、Enterで次へ）。
- **🔁 間隔反復（Leitnerボックス方式）** — 間違えた問題ほど短い間隔で、覚えた問題は長い間隔で自動的に復習キューへ。ナビの「復習」に出題予定数が表示されます。
- **📝 模擬試験** — 全分野または特定分野から、10〜60問をランダム出題。
- **📊 進捗ダッシュボード** — 分野ごとの習熟度・正答率・復習待ち数を一覧表示。
- **📕 用語集** — 全895語を横断検索。
- **🌙 ダーク/ライトテーマ切替**

## 学習の進め方（おすすめ）
1. 「基礎・方法論」から順に **学ぶ → 演習する** を回す。
2. 毎回まず **「復習」** で出題予定の問題を片付ける（間隔反復で長期記憶へ）。
3. 各分野がひととおり済んだら **「模擬試験（全分野）」** で実力チェック。

## ファイル構成
```
index.html              … エントリポイント（各分野スクリプトを読み込み）
assets/css/style.css    … デザイン
assets/js/app.js        … 共通エンジン（ルーター / クイズ / 間隔反復 / 進捗 / 用語集 / 模擬試験）
data/<分野>.js          … 各分野のコンテンツ（PsychApp.registerModule で自己登録）
```
新しい分野や問題を足したいときは、`data/intro.js` を雛形に `data/` へ `.js` を追加し、`index.html` の `<script>` に1行加えるだけです。

学習データのリセットは「進捗」ページ下部のボタンから行えます。
