Level 1 レッスン 1-6

モンティ・ホール問題と迷惑メールフィルタ

このレッスンでできるようになること

  • モンティ・ホール問題をベイズの定理で解き、扉を変えるべき理由を説明できる
  • 迷惑メールフィルタが単語の尤度から事後確率を計算するしくみを説明できる
  • 身近な問題を「仮説・データ・尤度」に翻訳して、定理を道具として使える

前提: 1-3 ベイズの定理の導出 と 1-4 事前確率・尤度・事後確率 を読んでいること

道具を手に、有名問題へ

ここまでで、ベイズの定理という道具が手に入りました。前回（1-5）は検査の問題で「直感がだまされる」体験をしましたね。

今回はその道具を、もっと使い倒します。テレビ番組で生まれたモンティ・ホール問題と、あなたのメールボックスを守る迷惑メールフィルタ。一見まったく別物ですが、どちらも同じ式で解けます。

新しい理論は出てきません。すでに知っている定理を、ちがう服を着た問題に当てはめる練習回です。気楽にいきましょう。

モンティ・ホール問題

アメリカのクイズ番組で、こんなゲームがありました。司会者の名前からモンティ・ホール問題と呼ばれています。

  • 3つの扉があります。1つの後ろには車、残り2つの後ろにはヤギ。
  • あなたは1つの扉を選びます（たとえば扉A）。
  • 司会者は、残りの扉のうちハズレ（ヤギ）の扉を1つ開けます。司会者は中身を知っていて、必ずヤギの扉を開けます。
  • あなたに選択権が与えられます。「最初の扉のまま」か「もう一方の閉じた扉に変える」か。

さて、車を当てたいあなたは、扉を変えるべきでしょうか。それとも変えないほうがいい？　多くの人は「残り2つなら五分五分、どっちでも同じ」と感じます。でも、そこに落とし穴があります。

直感でつかむ

カギは「司会者はわざとハズレを開けている」ことです。司会者は中身を知っていて、ヒントを漏らしているのです。

最初に選んだ扉が当たりの確率は \(1/3\) です。これは司会者が何をしても変わりません。すると、残り全体が当たりの確率は \(2/3\) のまま。司会者がハズレを1つつぶしてくれたので、その \(2/3\) が「もう一方の扉」に集まります。だから扉を変えると当たる確率は \(2/3\) になります。

言葉だけでは納得しづらいですよね。きちんとベイズの定理で確かめましょう。

問題を仮説とデータに翻訳する

あなたが扉Aを選んだとします。当たりがどこかが仮説、司会者がどの扉を開けたかがデータです。司会者が扉Bを開けた場合を考えましょう。

  • 仮説 \(A\): 車は扉Aの後ろ（最初の選択が当たり）
  • 仮説 \(C\): 車は扉Cの後ろ（もう一方が当たり）
  • データ \(D\): 司会者が扉Bを開けた

知りたいのは「データを見た後で、車が扉Cにある確率」つまり \(P(C \mid D)\) です。\(P(C \mid D)\) は『Dが起きたもとでのCの確率』と読みます。

まず事前確率（じぜんかくりつ。データを見る前の確率）は、3つの扉が同じなので次の通りです。

事前確率（どの扉も平等） $$ P(A) = P(B) = P(C) = \frac{1}{3} $$

次に尤度（ゆうど。その仮説のもとでデータが起きる確率）を考えます。「車がそこにあると仮定したら、司会者が扉Bを開ける確率はいくつか」を一つずつ見ます。

  • 仮説 \(A\) が正しいとき（車はA、あなたの選択が当たり）: 司会者は残るBとCのどちらもヤギなので、自由に選べます。Bを開ける確率は \(P(D \mid A) = \dfrac{1}{2}\)。
  • 仮説 \(C\) が正しいとき（車はC）: 司会者はあなたの扉Aも当たりの扉Cも開けられません。開けられるのはBだけ。だから \(P(D \mid C) = 1\)。
例題 — ベイズの定理で計算する

上の事前確率と尤度から、\(P(C \mid D)\)（扉を変えると当たる確率）を求めましょう。

解: まず分母の \(P(D)\)（司会者がBを開ける全体の確率）を求めます。Bが当たりだと司会者はBを開けられないので、関係するのは仮説 \(A\) と \(C\) です。

$$ P(D) = P(D \mid A)P(A) + P(D \mid C)P(C) = \frac{1}{2}\cdot\frac{1}{3} + 1\cdot\frac{1}{3} = \frac{1}{6} + \frac{2}{6} = \frac{3}{6} = \frac{1}{2} $$

これをベイズの定理に入れます。

$$ P(C \mid D) = \frac{P(D \mid C)\,P(C)}{P(D)} = \frac{1 \cdot \frac{1}{3}}{\frac{1}{2}} = \frac{\frac{1}{3}}{\frac{1}{2}} = \frac{2}{3} $$

同じように最初の扉のままの確率は \(P(A \mid D) = \dfrac{\frac{1}{2}\cdot\frac{1}{3}}{\frac{1}{2}} = \dfrac{1}{3}\)。

結論: 扉を変えると当たる確率は \(2/3\)、変えないと \(1/3\)。扉は変えるべきです。直感が \(1/2\) と感じたのは、司会者のヒントを無視していたからでした。

つまずき注意

「残り2つだから五分五分」は、司会者がランダムに扉を開けるなら正しい話です。モンティ・ホール問題が特別なのは、司会者が中身を知っていて必ずハズレを開けるから。司会者の行動が尤度に効き、確率を偏らせます。ルールが少し変わるだけで答えが変わる、いい例です。

問: モンティ・ホール問題で、扉を変えると当たる確率はいくつ？

正解は \(2/3\) です。最初の選択が当たる確率は \(1/3\) のまま変わりません。残りの \(2/3\) が、司会者がハズレを開けたことで「もう一方の扉」に集まります。ベイズの定理で計算しても \(P(C \mid D) = \dfrac{1 \cdot \frac{1}{3}}{1/2} = \dfrac{2}{3}\) となります。

迷惑メールフィルタ

次は実用例です。メールソフトは、届いたメールが迷惑メール（スパム）かどうかを自動で判定しています。その心臓部にもベイズの定理が使われています。

考え方はこうです。「当選」「無料」といった単語は、迷惑メールによく出てきます。逆にふつうのメールにはあまり出ません。この「単語の出やすさの差」が尤度になります。

直感でつかむ

あなたは経験的に「『無料』って書いてあるメールは怪しいな」と感じますよね。それを数で表すだけです。

迷惑メールでは「無料」が出やすく、ふつうのメールでは出にくい。すると「無料」という単語は、そのメールが迷惑メールだという証拠になります。ベイズの定理は、こうした証拠を集めて事前確率を更新し、事後確率（迷惑メールである確率）を出してくれます。

仮説とデータに翻訳しましょう。

仮説とデータ
  • 仮説 \(S\): そのメールは迷惑メール（spam）
  • 仮説 \(H\): そのメールはふつうのメール（ham と呼びます）
  • データ \(W\): メールに「無料」という単語が含まれている

知りたいのは \(P(S \mid W)\)、つまり「『無料』が入っているメールが迷惑メールである確率」です。

過去のメールを集計して、次の数字がわかったとします。

「無料」あり「無料」なし合計
迷惑メール (S)24060300
ふつう (H)70630700
合計3106901,000

この表から、必要な確率を読み取ります。

  • 事前確率: 全メールのうち迷惑メールは \(P(S) = \dfrac{300}{1000} = 0.3\)、ふつうは \(P(H) = 0.7\)。
  • 尤度: 迷惑メールが「無料」を含む確率 \(P(W \mid S) = \dfrac{240}{300} = 0.8\)。ふつうのメールが含む確率 \(P(W \mid H) = \dfrac{70}{700} = 0.1\)。
例題 — 「無料」を見て確率を更新する

「無料」を含むメールが届きました。これが迷惑メールである確率 \(P(S \mid W)\) を求めましょう。

解: まず分母 \(P(W)\)（どんなメールでも「無料」を含む確率）を求めます。

$$ P(W) = P(W \mid S)P(S) + P(W \mid H)P(H) = 0.8 \times 0.3 + 0.1 \times 0.7 = 0.24 + 0.07 = 0.31 $$

ベイズの定理に入れます。

$$ P(S \mid W) = \frac{P(W \mid S)\,P(S)}{P(W)} = \frac{0.8 \times 0.3}{0.31} = \frac{0.24}{0.31} \approx 0.77 $$

結論: 事前は \(30\%\) だった「迷惑メールらしさ」が、「無料」という単語1つで約 \(77\%\) まで跳ね上がりました。表の「無料あり」列で確かめると \(240 / 310 \approx 0.77\) と一致します。これがフィルタが行っている計算です。

つまずき注意 — 尤度と事後確率はちがう

\(P(W \mid S) = 0.8\)（迷惑メールが「無料」を含む確率）と、\(P(S \mid W) \approx 0.77\)（「無料」を含むメールが迷惑メールである確率）は別物です。向きが逆ですね。実際この問題では値も少しちがいます。\(P(A \mid B)\) と \(P(B \mid A)\) を取りちがえないこと。ベイズの定理は、この2つを正しく行き来させる橋でした。

単語が増えたらどうする？

実際のフィルタは「無料」だけでなく「当選」「クリック」など、たくさんの単語を同時に見ます。そのとき、各単語が独立だと割り切って、尤度をかけ算でつなげていきます。たとえば最初の「無料」で得た事後確率を、次の単語を見るときの事前確率として使い、次々と更新していくのです。

この「単語どうしは独立とみなしてかけ算する」割り切った方法をナイーブベイズ（素朴ベイズ）と呼びます。本当は単語どうしに関係があるのに「ない」とみなすので素朴（ナイーブ）なのですが、これがおどろくほどよく効きます。

やってみよう

上の例で、もし「無料」に続いて2つ目の怪しい単語が見つかったら、確率はどうなるでしょう。約 \(0.77\) を新しい事前確率にして、2つ目の単語の尤度（迷惑なら出やすく、ふつうなら出にくい）をかければ、確率はさらに \(1\) に近づきます。逆にふつうのメールに多い単語が見つかれば下がります。証拠を1つずつ積み上げて信念を更新する、これがベイズの世界観です。

くわしいナイーブベイズの数式やゼロ頻度の扱いは Level 5 で本格的に扱います。ここでは「単語の尤度を集めて事後確率を出す」という骨組みがつかめれば十分です。

問: 「無料」を含むメールの迷惑メール確率が \(0.77\) と求まりました。この計算で「データ」にあたるのはどれ？

正解はCです。ベイズの定理では「観測した事実」がデータです。ここでは「『無料』が含まれている」という観測がデータ \(W\) で、求めたい仮説 \(S\)（迷惑メールである）の確率を更新する材料になります。Aは仮説、Bは事前確率です。

演習

演習1 — モンティ・ホール、扉が増えたら

扉が100個あり、当たりは1つだけ。あなたが1つ選んだあと、司会者は残り99個のうち98個のハズレを開けます。残ったのはあなたの扉と、もう1つの扉だけ。扉を変えると当たる確率はいくつでしょう。

解答を見る

最初に選んだ扉が当たる確率は \(\dfrac{1}{100}\)。これは変わりません。残り99個全体が当たりの確率は \(\dfrac{99}{100}\) で、司会者が98個のハズレを開けてつぶしたので、その \(\dfrac{99}{100}\) がまるごと「もう1つの扉」に集まります。よって扉を変えると当たる確率は \(\dfrac{99}{100}\)。扉が多いほど、変えるべき理由がはっきりします。3扉のときの \(2/3\) も同じ理屈です。

演習2 — 別の単語で計算する

あるメールサービスで、迷惑メールは全体の \(40\%\)（\(P(S)=0.4\)、\(P(H)=0.6\)）。単語「当選」について、\(P(\text{当選} \mid S) = 0.5\)、\(P(\text{当選} \mid H) = 0.02\) でした。「当選」を含むメールが迷惑メールである確率 \(P(S \mid \text{当選})\) を求めなさい。

解答を見る

分母 \(P(\text{当選})\) を求めます。

$$ P(\text{当選}) = 0.5 \times 0.4 + 0.02 \times 0.6 = 0.20 + 0.012 = 0.212 $$

ベイズの定理に入れます。

$$ P(S \mid \text{当選}) = \frac{0.5 \times 0.4}{0.212} = \frac{0.20}{0.212} \approx 0.94 $$

約 \(94\%\)。「当選」はふつうのメールにほとんど出ない（\(0.02\)）ぶん、強い証拠になります。ふつうのメールで出にくい単語ほど、見つかったときに確率を大きく動かすのがポイントです。

演習3 — 司会者がランダムだったら

モンティ・ホール問題で、もし司会者が中身を知らず、残り2つの扉からでたらめに開けたとします（たまたまヤギが出た場合を考える）。このとき扉を変えると当たる確率はいくつ？　ベイズの定理で確かめなさい。

解答を見る

あなたが扉Aを選び、司会者がでたらめに扉Bを開けてヤギが出た、という状況です。尤度が変わります。司会者は中身を知らないので、Bを開ける確率はどの仮説でも \(1/2\)。ただしBが当たりだと「ヤギが出た」という条件に合わないので、仮説 \(A\) と \(C\) を比べます。

\(P(D \mid A) = \dfrac{1}{2}\)（Bを選ぶ、中身はヤギ）、\(P(D \mid C) = \dfrac{1}{2}\)（Bを選ぶ、中身はヤギ）。事前はどちらも \(1/3\) なので、

$$ P(C \mid D) = \frac{\frac{1}{2}\cdot\frac{1}{3}}{\frac{1}{2}\cdot\frac{1}{3} + \frac{1}{2}\cdot\frac{1}{3}} = \frac{1}{2} $$

今度は五分五分です。司会者がわざとハズレを開けるという情報がないと、ヒントが消えてしまう。だから本物のモンティ・ホール問題では「司会者は中身を知っている」というルールが決定的に効くのです。

まとめ

  • モンティ・ホール問題では、扉を変えると当たる確率は \(2/3\)。司会者が必ずハズレを開けるという行動が尤度に効くからです。
  • 司会者がでたらめに開けるなら五分五分。ルール（＝尤度）が変われば答えも変わります。
  • 迷惑メールフィルタは、単語の尤度（迷惑なら出やすい・ふつうなら出にくい）から事後確率を計算する装置です。
  • 「無料」のような単語1つで、事前 \(30\%\) が事後 \(77\%\) へ。証拠を見て信念を更新するのがベイズです。
  • 単語を独立とみなしてかけ算で更新する方法をナイーブベイズと呼びます。くわしくは Level 5 で。

どんな問題も「仮説・データ・尤度」に翻訳できれば、あとは同じ式に当てはめるだけ。次のレベルからは、点の確率ではなく「分布」を相手にして、ベイズの世界をぐっと広げていきます。
