Level 1 レッスン 1-1

条件付き確率

このレッスンでできるようになること

  • 条件付き確率 \(P(A \mid B)\) を、定義式と分割表の両方から計算できる
  • \(P(A \mid B)\) を「BのもとでのAの確率」と正しく読める
  • \(P(A \mid B)\) と \(P(B \mid A)\) が別物であることを、例を挙げて説明できる

前提: 0-4 表で整理する確率 と 0-5 「逆向き」の確率 — ベイズへの入り口 を読んでいること

「わかったとき」で確率は変わる

朝、家を出るときのことを考えてみましょう。「今日、雨が降る確率は？」と聞かれたら、なんとなく答えますよね。

では「空が真っ黒な雲でおおわれているとわかったとき、雨が降る確率は？」だとどうでしょう。さっきよりずっと高く感じるはずです。

同じ「雨が降る」でも、何か情報を知ったかどうかで確率は変わります。この「〜だとわかったときの確率」を、きちんと数で表す道具が、このレッスンの主役です。

直感でつかむ

確率は「全体の中での割合」でした。条件付き確率がやっているのは、全体の見る範囲をしぼることだけです。

「雨が降る割合」はカレンダー全部の日が母数です。でも「黒い雲が出ている日にしぼると、そのうち雨が降る割合は？」と聞かれたら、母数は黒い雲の日だけに縮みます。情報を知るとは、母数を縮めることなのです。

条件付き確率の定義

事象（できごと）に名前をつけます。\(A\)、\(B\) のように大文字で書くのが約束です。たとえば「雨が降る」を \(A\)、「黒い雲が出ている」を \(B\) とします。

このとき「\(B\) が起きたとわかったうえで \(A\) が起きる確率」を 条件付き確率（じょうけんつきかくりつ）と呼び、\(P(A \mid B)\) と書きます。縦棒 \(\mid\) の右が「わかっていること（条件）」です。

\(P(A \mid B)\) は 「BのもとでのAの確率」 と読みます。英語では「A given B」と言い、given（〜のもとで）の後ろが条件 \(B\) になります。縦棒の右が条件、という約束とちょうど対応しているわけです。

定義 — 条件付き確率

事象 \(B\) が起きたとわかったときに事象 \(A\) が起きる確率 \(P(A \mid B)\) を、次の式で定義します（ただし \(P(B) > 0\) とします）: $$ P(A \mid B) = \frac{P(A \cap B)}{P(B)} $$ ここで \(P(A \cap B)\) は「\(A\) と \(B\) が両方とも起きる確率」です。\(\cap\) は「かつ」を表す記号で、「キャップ」と読みます。

式の気持ちを言葉にしてみます。下の図のように、母数を \(B\) の中だけにしぼり、その中で \(A\) も起きている部分の割合を見ているのです。

分割表で具体的に

0-4 で学んだ分割表（クロス集計表）を使うと、条件付き確率は実物として見えてきます。ある町の100人に「黒い雲が出ているか」「雨が降ったか」を記録したとしましょう。

表では、ある事象が起きないことを、文字の上に横棒をつけて表します。たとえば \(\bar A\)（エー・バー）は「\(A\) が起きない＝雨が降らない」、\(\bar B\) は「\(B\) が起きない＝黒い雲が出ていない」という意味です。

雨が降った (\(A\))雨は降らない (\(\bar A\))合計
黒い雲あり (\(B\))241640
黒い雲なし (\(\bar B\))65460
合計3070100

ここで「黒い雲が出ているとわかったとき、雨が降る確率」、つまり \(P(A \mid B)\) を求めます。コツは、表の「黒い雲あり」の行だけを見ることです。

黒い雲ありの日は全部で40日。そのうち雨が降ったのは24日。だから割合は次のとおりです。

分割表から直接 $$ P(A \mid B) = \frac{\text{両方起きた人数}}{\text{Bの人数}} = \frac{24}{40} = 0.6 $$

定義式 \(P(A \mid B) = \dfrac{P(A \cap B)}{P(B)}\) と同じことを確かめましょう。\(P(A \cap B) = \dfrac{24}{100}\)、\(P(B) = \dfrac{40}{100}\) なので、

$$ P(A \mid B) = \frac{24/100}{40/100} = \frac{24}{40} = 0.6 $$

分母どうしの100が約分で消えて、結局「人数どうしの割り算」になります。だから分割表では、条件の行（または列）だけ取り出して割るだけでよいのです。

直感でつかむ

ちなみに、情報を知る前の「ふつうに雨が降る確率」は \(P(A) = \dfrac{30}{100} = 0.3\) でした。黒い雲を見たことで \(0.3\) が \(0.6\) に上がりました。「わかったこと」が確率を引き上げたわけです。これがベイズの世界の出発点です。

例題

例題

上と同じ100人の表を使います。今度は逆に「雨が降ったとわかったとき、その日に黒い雲が出ていた確率」、つまり \(P(B \mid A)\) を求めましょう。

解: 今度は条件が「雨が降った (\(A\))」なので、表の「雨が降った」の列だけを見ます。雨が降ったのは合計30日。そのうち黒い雲が出ていたのは24日です。よって、

$$ P(B \mid A) = \frac{P(A \cap B)}{P(A)} = \frac{24/100}{30/100} = \frac{24}{30} = 0.8 $$

同じ「24」という人数を使っているのに、割る相手（母数）が \(40\) から \(30\) に変わったため、答えは \(0.6\) ではなく \(0.8\) になりました。次の節で見るように、この2つは別物です。

P(A|B) と P(B|A) は別物

例題でわかったとおり、\(P(A \mid B) = 0.6\)、\(P(B \mid A) = 0.8\) で、値がちがいました。これは偶然ではありません。「縦棒の左右を入れかえると、ふつう答えは変わる」のです。

理由は定義式を見れば一目です。分子の \(P(A \cap B)\) は左右を入れかえても同じですが、分母が変わります。\(P(A \mid B)\) は \(P(B)\) で割り、\(P(B \mid A)\) は \(P(A)\) で割ります。割る相手がちがえば、答えもちがって当然です。

つまずき注意

\(P(A \mid B)\) と \(P(B \mid A)\) は一般に等しくありません。これはベイズ統計でいちばん多いつまずきです。

身近な例で言うと、「雨の日に傘を持っている人の割合」と「傘を持っている人のうち雨の日の割合」はまったくの別物です。雨でなくても傘を持つ人はいますから。\(\mid\) の右と左を取りちがえると、結論が大きくずれます。読むときは必ず「どっちが条件か」を先に確認しましょう。

理解度チェック

問1: \(P(A \mid B)\) はどう読みますか？

縦棒 \(\mid\) の右が条件です。右にある \(B\) が「わかっていること」なので、「BのもとでのAの確率」と読みます。なお「AとBが両方起きる確率」は \(P(A \cap B)\) であり、これとは別物です。

問2: 本文の100人の表で、「黒い雲が出ていないとわかったとき、雨が降る確率」 \(P(A \mid \bar B)\) はいくつですか？

条件は「黒い雲なし (\(\bar B\))」なので、その行だけを見ます。黒い雲なしの日は合計60日、そのうち雨が降ったのは6日。よって \(\dfrac{6}{60} = 0.1\) です。母数は \(\bar B\) の人数（60）で、雨の合計（30）でも全体（100）でもないことに注意しましょう。

問3: \(P(A \mid B)\) と \(P(B \mid A)\) について正しいのはどれですか？

どちらも分子は \(P(A \cap B)\)（両方起きる確率）で共通です。ちがうのは分母で、\(P(A \mid B)\) は \(P(B)\)、\(P(B \mid A)\) は \(P(A)\) で割ります。分母がちがうので値も一般にちがいますが、同じ分子を共有している点で無関係ではありません。この「つながり」を式にしたのが、のちのち学ぶベイズの定理です。

演習

演習1

あるクラス40人に、犬が好きか猫が好きかを聞きました。結果は次の表のとおりです。「猫が好きとわかったとき、その人が犬も好きな確率」 \(P(\text{犬} \mid \text{猫})\) を求めなさい。

犬が好き犬は好きでない合計
猫が好き9615
猫は好きでない16925
合計251540
解答を見る

条件は「猫が好き」なので、その行だけを見ます。猫好きは合計15人、そのうち犬も好きなのは9人。よって、 $$ P(\text{犬} \mid \text{猫}) = \frac{9}{15} = 0.6 $$ です。母数は猫好きの15人であって、全体の40人でも犬好きの25人でもありません。

演習2

演習1と同じ表で、今度は \(P(\text{猫} \mid \text{犬})\)（犬が好きとわかったとき、猫も好きな確率）を求めなさい。演習1の答えと比べて、何が言えますか。

解答を見る

条件が「犬が好き」に変わったので、今度は列を見ます。犬好きは合計25人、そのうち猫も好きなのは9人。よって、 $$ P(\text{猫} \mid \text{犬}) = \frac{9}{25} = 0.36 $$ です。演習1では \(P(\text{犬} \mid \text{猫}) = 0.6\) でしたから、縦棒の左右を入れかえると答えが変わりました（\(0.6 \ne 0.36\)）。分子はどちらも「両方好き」の9人で共通ですが、割る母数が \(15\) と \(25\) でちがうためです。

演習3

くじが10本あり、当たりは3本です。1本引いたとき、それが当たりである事象を \(A\) とします。「引いたくじが赤色だった」という情報を \(B\) とし、赤いくじは4本、そのうち当たりは2本だとします。\(P(A \mid B)\) を求めなさい。

解答を見る

条件は「赤色だった (\(B\))」なので、母数は赤いくじ4本にしぼられます。そのうち当たりは2本。よって、 $$ P(A \mid B) = \frac{2}{4} = 0.5 $$ です。情報を知る前の当たりの確率は \(P(A) = \dfrac{3}{10} = 0.3\) でしたが、「赤だった」とわかったことで \(0.5\) に上がりました。全体10本ではなく、条件にあう4本で割るのがポイントです。

演習4

確率の言葉で答えなさい。\(P(A \cap B) = 0.2\)、\(P(B) = 0.5\) のとき、\(P(A \mid B)\) はいくつですか。

解答を見る

定義式にそのまま当てはめます。 $$ P(A \mid B) = \frac{P(A \cap B)}{P(B)} = \frac{0.2}{0.5} = 0.4 $$ 分割表がなくても、\(P(A \cap B)\) と \(P(B)\) の2つの値さえあれば、割り算ひとつで条件付き確率は求まります。

まとめ

  • 条件付き確率 \(P(A \mid B)\) は「BのもとでのAの確率」と読み、縦棒の右が条件。
  • 定義は \(P(A \mid B) = \dfrac{P(A \cap B)}{P(B)}\)。母数を \(B\) の世界にしぼった割合のこと。
  • 分割表では、条件の行（または列）だけを取り出して割るだけで求まる。
  • 情報を知ると母数が縮み、確率は上がることも下がることもある。
  • \(P(A \mid B)\) と \(P(B \mid A)\) は分母がちがうので、一般に別物。取りちがえに注意。

「左右を入れかえると別物」という今日のひっかかりは、じつは次のレッスンで大活躍します。次は 1-2 同時確率と周辺確率 で、\(P(A \cap B)\) と \(P(A)\) の関係を整理しましょう。これがベイズの定理への確かな一歩になります。
