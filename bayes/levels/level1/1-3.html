Level 1 レッスン 1-3

ベイズの定理の導出

このレッスンでできるようになること

  • 同時確率を2通りに書くことから、ベイズの定理を自分の手で導ける
  • ベイズの定理の各部分が何を表すかを説明できる
  • 全確率の定理を使って、分母 \(P(B)\) を場合分けで展開できる

前提: 1-1 条件付き確率 と 1-2 同時確率と周辺確率 を読んでいること

「逆向き」を計算したい

検査を受けて陽性が出たとします。知りたいのは「陽性だったとき、本当に病気である確率」ですよね。 でも、手元にある資料に書いてあるのはたいてい逆の数字です。 「病気の人が検査を受けたら、何％が陽性になるか」です。

この2つは向きが反対です。0-5「逆向き」の確率 でも見たように、向きを取りちがえると大きな誤解が生まれます。 欲しい向きと、手元にある向き。この橋渡しをしてくれるのが、これから導くベイズの定理（ていり）です。

うれしいことに、この定理は天から降ってくる魔法ではありません。 1-1 で習った条件付き確率の定義だけを使って、自分で組み立てられます。 このレッスンでは、その組み立てを最初から最後まで一緒にたどります。

直感でつかむ

ある同じ部屋（2つの出来事が両方起きた状況）に入るのに、ドアが2つあると思ってください。

1つ目のドアは「まず \(B\) を確かめて、そのあと \(A\) を確かめる」道。 2つ目は「まず \(A\) を確かめて、そのあと \(B\) を確かめる」道。 入口は違っても、たどり着く部屋は同じ「\(A\) も \(B\) も起きた」です。

つまり2つの道の「通りやすさ」は等しいはず。この当たり前の等号から、ベイズの定理が一気に出てきます。

出発点: 同時確率を2通りに書く

まず 1-1 の条件付き確率（じょうけんつきかくりつ）の定義を思い出します。 \(P(A \mid B)\) は「\(B\) が起きたとわかったときの \(A\) の確率」と読みます。定義はこうでした。

定義 — 条件付き確率（復習）

\(P(B) > 0\) のとき、 $$ P(A \mid B) = \frac{P(A \cap B)}{P(B)} $$ ここで \(P(A \cap B)\) は「\(A\) と \(B\) が両方起きる確率」＝同時確率（どうじかくりつ）です（1-2 で学習）。

この式の両辺に \(P(B)\) をかけると、分母が消えて、同時確率を表す式になります。

同時確率の書き方 その1（B から見る） $$ P(A \cap B) = P(A \mid B)\, P(B) $$

これは「先に \(B\) が起きて（確率 \(P(B)\)）、その \(B\) のもとで \(A\) も起きる（確率 \(P(A \mid B)\)）」という読み方です。さきほどの1つ目のドアですね。

ところで、\(A\) と \(B\) の役割を入れかえても、まったく同じ理屈が成り立ちます。 条件付き確率の定義で \(A\) と \(B\) を交換すると \(P(B \mid A) = \dfrac{P(A \cap B)}{P(A)}\) となり、両辺に \(P(A)\) をかけて、

同時確率の書き方 その2（A から見る） $$ P(A \cap B) = P(B \mid A)\, P(A) $$

こちらは2つ目のドア。「先に \(A\) が起きて、その \(A\) のもとで \(B\) も起きる」という読み方です。 なお \(A \cap B\) と \(B \cap A\) は同じ事象なので、左辺はどちらも同じ \(P(A \cap B)\) です。

2つを等号でつなぐ

ここがヤマ場です。「その1」も「その2」も、左辺は同じ \(P(A \cap B)\) でした。 同じものを表す式が2つあるなら、右辺どうしも等しいはずです。

2通りの書き方は等しい $$ P(A \mid B)\, P(B) = P(B \mid A)\, P(A) $$

あとは知りたい向き \(P(A \mid B)\) について解くだけ。両辺を \(P(B)\) で割ります（\(P(B) > 0\) が必要です）。 すると分母に \(P(B)\) が移り、ベイズの定理が完成します。

定理 — ベイズの定理

\(P(B) > 0\) のとき、 $$ P(A \mid B) = \frac{P(B \mid A)\, P(A)}{P(B)} $$ 手元にある向き \(P(B \mid A)\) から、知りたい向き \(P(A \mid B)\) を計算できる、というのが核心です。

導出をもう一度ひとことで言うと、こうです。 同時確率を2通りに書いて、等号で結んで、欲しい向きについて解いた。 たったこれだけです。新しく覚えることは何もなく、1-1 の定義から自然に出てきました。

つまずき注意

ベイズの定理は \(P(A \mid B)\) と \(P(B \mid A)\) を「等しい」とする式ではありません。 むしろ、この2つは一般に等しくないからこそ、向きを変換する道具が必要なのです。 右辺をよく見ると \(P(A)\) と \(P(B)\) という重みがかかっていますね。この重みのぶんだけ、2つの向きはズレます。

分母 P(B) を場合分けで展開する

実際に計算しようとすると、分母の \(P(B)\) が手元の資料に書いていない、ということがよくあります。 たとえば「陽性になる確率」そのものは、ふつう資料に載っていません。 そこで \(P(B)\) を、自分が持っている数字だけで組み立て直します。これに使うのが全確率の定理（ぜんかくりつのていり）です。

直感でつかむ

「陽性になる」には2つの入口があります。「病気で、かつ陽性」と「病気でなく、かつ陽性」です。 この2つで陽性のすべてを覆い、しかも重なりません。 だから「陽性になる確率」は、2つの入口の確率を足し合わせたものになります。

いま、世界を \(A\)（起きた）と \(\bar{A}\)（起きなかった。「A バー」と読みます）の2つに分けます。 どんな場合でも、必ずどちらか一方に入り、両方に入ることはありません。 すると事象 \(B\) は、「\(A\) と一緒に起きた \(B\)」と「\(\bar{A}\) と一緒に起きた \(B\)」の2つに切り分けられます。

この2つの片は重なりません。だから「\(B\) が起きる確率」は、2つの片の確率の単純な足し算です。

B を2つの片に分ける $$ P(B) = P(A \cap B) + P(\bar{A} \cap B) $$

さらに、それぞれの片を「その1」の書き方（同時確率＝条件付き確率 × 確率）で書き直します。 \(P(A \cap B) = P(B \mid A)\,P(A)\)、\(P(\bar{A} \cap B) = P(B \mid \bar{A})\,P(\bar{A})\) ですから、

定理 — 全確率の定理（2分割の場合）

$$ P(B) = P(B \mid A)\, P(A) + P(B \mid \bar{A})\, P(\bar{A}) $$ 右辺はすべて「手元にありそうな数字」だけでできています。これを分母に入れれば計算できます。

このひらいた \(P(B)\) をベイズの定理の分母に代入すると、分母を場合分けで展開した完成形になります。

ベイズの定理（分母を展開した完成形） $$ P(A \mid B) = \frac{P(B \mid A)\, P(A)}{P(B \mid A)\, P(A) + P(B \mid \bar{A})\, P(\bar{A})} $$
分子: A の側から B が起きる「道の太さ」 分母: B が起きる全部の道（A 側 ＋ Ā 側） 全体: B のうち A の道が占める割合

見た目はいかつくなりましたが、やっていることは「\(B\) という全体のうち、\(A\) 経由の分がどれだけか」という割合の計算です。 分子は分母の一部になっていますね。

例題で通してみる

例題

ある病気にかかっている人は、住民全体の \(1\%\) です。検査には次のクセがあります。

  • 病気の人が検査を受けると、\(90\%\) で陽性になる（＝ \(P(\text{陽性} \mid \text{病気}) = 0.9\)）
  • 病気でない人でも、\(5\%\) で陽性になってしまう（＝ \(P(\text{陽性} \mid \text{健康}) = 0.05\)）

いま、ある人が陽性でした。この人が本当に病気である確率 \(P(\text{病気} \mid \text{陽性})\) を求めましょう。

解：\(A=\)「病気」、\(B=\)「陽性」とおきます。手元の数字は次の通り。

\(P(A)=0.01\)、\(P(\bar{A})=0.99\)、\(P(B \mid A)=0.9\)、\(P(B \mid \bar{A})=0.05\)。

まず分母（陽性になる確率）を全確率の定理でひらきます。

$$ P(B) = 0.9 \times 0.01 + 0.05 \times 0.99 = 0.009 + 0.0495 = 0.0585 $$

つぎにベイズの定理に入れます。

$$ P(A \mid B) = \frac{0.9 \times 0.01}{0.0585} = \frac{0.009}{0.0585} \approx 0.154 $$

つまり、陽性でも病気である確率は約 \(15\%\) です。 「\(90\%\) で陽性になる検査」と聞くと高そうですが、もともと病気の人がごく少ない（\(1\%\)）ため、 陽性の多くは健康な人の「まちがい陽性」で占められます。だから \(15\%\) にとどまるのです。

理解度チェック

問1: ベイズの定理を導く出発点になった等式はどれですか?

同時確率 \(P(A \cap B)\) を2通りに書き、左辺が同じことから右辺どうしを等号で結んだのが \(P(A \mid B)\,P(B) = P(B \mid A)\,P(A)\) です。これを \(P(B)\) で割ると定理になります。aは一般に成り立たず、cは独立のときだけ成り立つ別の式です。

問2: ベイズの定理 \(P(A \mid B) = \dfrac{P(B \mid A)\,P(A)}{P(B)}\) の分母 \(P(B)\) を、全確率の定理でひらくとどうなりますか?

\(B\) を「\(A\) と一緒の片」と「\(\bar{A}\) と一緒の片」に分け、それぞれを同時確率の書き方で表すと \(P(B \mid A)\,P(A) + P(B \mid \bar{A})\,P(\bar{A})\) です。各条件付き確率には、その条件の確率（\(P(A)\) や \(P(\bar{A})\)）という重みをかけ忘れないのがポイントです。

問3: 例題と同じ設定で、もし病気の割合 \(P(A)\) が \(1\%\) ではなく \(50\%\) だったら、陽性のときに病気である確率はどう変わりますか?

事前確率 \(P(A)\) が上がると、分子 \(P(B \mid A)\,P(A)\) が増え、まちがい陽性の割合は相対的に減ります。実際 \(P(A)=0.5\) なら \(P(B)=0.9 \times 0.5 + 0.05 \times 0.5 = 0.475\)、\(P(A \mid B)=\dfrac{0.45}{0.475} \approx 0.95\)。もとが少ないか多いかで、同じ検査でも結論が大きく変わります。

演習

演習1

条件付き確率の定義 \(P(A \mid B) = \dfrac{P(A \cap B)}{P(B)}\) から出発して、 ベイズの定理 \(P(A \mid B) = \dfrac{P(B \mid A)\,P(A)}{P(B)}\) を、途中式つきで導きなさい。

解答を見る

(1) 定義の両辺に \(P(B)\) をかけて \(P(A \cap B) = P(A \mid B)\,P(B)\)。

(2) \(A\) と \(B\) を入れかえた定義から \(P(A \cap B) = P(B \mid A)\,P(A)\)。（\(A \cap B\) と \(B \cap A\) は同じ事象）

(3) 左辺が同じなので \(P(A \mid B)\,P(B) = P(B \mid A)\,P(A)\)。

(4) 両辺を \(P(B)\) で割って \(P(A \mid B) = \dfrac{P(B \mid A)\,P(A)}{P(B)}\)。これがベイズの定理です。

演習2

ある工場では製品の \(2\%\) に傷があります。検査機は、傷のある製品を \(95\%\) の確率で「不良」と判定し、 傷のない製品も \(10\%\) の確率で「不良」と判定してしまいます。 「不良」と判定された製品が、本当に傷ありである確率を求めなさい。

解答を見る

\(A=\)「傷あり」、\(B=\)「不良判定」とおきます。\(P(A)=0.02\)、\(P(\bar{A})=0.98\)、\(P(B \mid A)=0.95\)、\(P(B \mid \bar{A})=0.10\)。

分母を全確率の定理でひらくと $$ P(B) = 0.95 \times 0.02 + 0.10 \times 0.98 = 0.019 + 0.098 = 0.117 $$

ベイズの定理に入れて $$ P(A \mid B) = \frac{0.95 \times 0.02}{0.117} = \frac{0.019}{0.117} \approx 0.162 $$

約 \(16\%\) です。「不良」と出ても、その多くは傷のない製品の取りちがえだとわかります。

演習3

次の文のまちがいを指摘し、正しい考え方を一言で書きなさい。 「この検査は病気の人を \(99\%\) 当てる。だから陽性なら病気である確率は \(99\%\) だ。」

解答を見る

\(99\%\) なのは \(P(\text{陽性} \mid \text{病気})\)、つまり「病気のもとで陽性になる」向きの確率です。 知りたいのは逆向きの \(P(\text{病気} \mid \text{陽性})\) で、この2つは一般に等しくありません。

正しくは、ベイズの定理で向きを変換する必要があります。 その際、病気の人がもともと少なければ（事前確率 \(P(\text{病気})\) が小さければ）、陽性でも病気である確率はずっと低くなります。 「当てる率」と「陽性なら病気の率」は別物、というのがポイントです。

まとめ

  • 同時確率 \(P(A \cap B)\) は2通りに書ける: \(P(A \mid B)\,P(B)\) と \(P(B \mid A)\,P(A)\)。
  • 左辺が同じなので右辺を等号で結び、\(P(B)\) で割るとベイズの定理 \(P(A \mid B) = \dfrac{P(B \mid A)\,P(A)}{P(B)}\) が出る。
  • 定理は \(P(A \mid B)\) と \(P(B \mid A)\) を等しいとする式ではない。向きを変換する道具である。
  • 分母 \(P(B)\) は全確率の定理で \(P(B \mid A)\,P(A) + P(B \mid \bar{A})\,P(\bar{A})\) と場合分けして展開できる。
  • 事前確率（もとの割合）が小さいと、検査が陽性でも本当に病気である確率は低くとどまる。

これでベイズの定理を自分の手で導けるようになりました。 ここに出てきた \(P(A)\)、\(P(B \mid A)\)、\(P(A \mid B)\) には、それぞれ事前確率・尤度（ゆうど）・事後確率という名前がついています。 次のレベルでは、この名前と意味をじっくり育てていきます。一段、のぼれましたね。
