用語集

このサイトに出てくるベイズ統計の用語を、五十音順に短くまとめました。 本文を読んでいて「あれ、この言葉なんだっけ?」となったら、ここを辞書のように引いてください。 各用語には、くわしく学べるレッスンへのリンクを付けてあります。

読み方は(かっこ)の中に書いています。 まずは太字の用語と一行の説明だけ読めば十分です。もっと知りたくなったら、右側の 該当レッスンのリンクへ進みましょう。記号(\(P(A\mid B)\)、\(\propto\) など)の読み方は 付録 a-4 記号表にまとめてあります。

あ か さ た な は ま や ら

あ行

MCMC(エムシーエムシー / マルコフ連鎖モンテカルロ) → 4-2
事後分布から直接サンプルを取り出せないとき、少しずつ動く乱数の散歩で分布を近似する手法群です。 散歩の足あと(サンプル)を集めると、その分布が事後分布の形になります。代表例がメトロポリス・ヘイスティングス法やHMCです。 くわしく: 4-3 メトロポリス・ヘイスティングス法 / 4-5 HMCとNUTS
MAP推定(マップすいてい / 最大事後確率推定) → 3-5
事後分布がいちばん高くなる点(山のてっぺん)を、推定値として一つ選ぶやり方です。 分布全体ではなく代表値が一つほしいときに使います。事前分布が平らなときは最尤推定と一致します。
事後分布(じご ぶんぷ) → 1-4
データを見た後のパラメータについての信念を表す分布です。事前分布に尤度を掛けて更新した結果がこれにあたります。 「結局このパラメータはどのあたりらしいか」を分布の形で答えてくれます。 分布としての扱い: 3-2 ベータ・二項モデル
事後確率(じご かくりつ) → 1-4
データを見た後の、ある仮説が正しい確率 \(P(H \mid D)\) です。ベイズの定理の左辺にあたります。 \(P(H \mid D)\) は「\(D\)(データ)が分かったときの \(H\)(仮説)の確率」と読みます。
事後予測分布(じご よそく ぶんぷ) → 5-7
推定し終えたモデルが「次のデータはこう出るだろう」と予言する分布です。 パラメータの不確かさも込みで予測するのが特徴で、モデルが妥当かを点検する事後予測チェックに使います。
事前分布(じぜん ぶんぷ) → 1-4
データを見る前に持っているパラメータについての信念を表す分布です。 「コインの表が出る確率はだいたい0.5あたりだろう」のような最初の見立てを、分布の形で書いたものです。 分布で表す例: 2-6 ベータ分布
事前確率(じぜん かくりつ) → 1-4
データを見る前の、ある仮説が正しい確率 \(P(H)\) です。ベイズの定理では尤度と掛け合わせる相手になります。

か行

階層ベイズモデル(かいそう ベイズ モデル) → 5-4
グループごとのパラメータが、さらに上位の分布から生まれると考えるモデルです。 「全体の傾向」と「各グループの個性」を同時に推定でき、データの少ないグループを全体の情報で支える部分プーリングが働きます。
期待値(きたいち) → 2-4
確率変数が平均してとる値です。とりうる値に、その確率という重みを掛けて足し合わせて求めます。 分布を一つの代表値で要約するときの基本量で、事後分布の点推定にも使われます。
共役事前分布(きょうやく じぜん ぶんぷ) → 3-3
ある尤度と組み合わせたとき、事後分布が事前分布と同じ種類の分布になるような事前分布です。 たとえば二項分布の尤度にはベータ分布が共役で、更新後もベータ分布のまま。計算が手で閉じるのが利点です。 例で体験: 3-2 ベータ・二項モデル
ギブスサンプリング(ギブス サンプリング) → 4-4
MCMCの一種で、パラメータを一つずつ「他を固定したときの条件付き分布」からサンプルし、順番に回していく方法です。 条件付き分布がきれいに書ける問題で力を発揮します。

さ行

最尤推定(さいゆう すいてい) → 2-5
尤度をいちばん大きくするパラメータを推定値として選ぶ方法です。ベイズではなく頻度論側の代表的な推定法ですが、 事前分布を平らにしたMAP推定と一致するので、ベイズとのつながりを知る入口になります。
収束診断(しゅうそく しんだん) → 4-6
MCMCのサンプルが本当に事後分布に行き着いたかを確かめる点検です。 trace plot(足あとの様子)・\(\hat{R}\)(複数の鎖のばらつき)・有効サンプルサイズなどを見て判断します。
周辺確率(しゅうへん かくりつ) → 1-2
片方の事象だけに注目した確率 \(P(A)\) です。分割表の端(マージン)の合計から得られることが名前の由来です。 同時確率を、もう一方の事象についてすべて足し上げると周辺確率になります。
周辺尤度(しゅうへん ゆうど / エビデンス) → 1-4
ベイズの定理の分母 \(P(D)\) で、考えられるすべての仮説を平均したデータの出やすさです。 事後分布の合計を1に整える役目があり、モデル比較ではベイズファクターの材料にもなります。 計算の壁: 4-1 解析的に解けない事後分布
条件付き確率(じょうけんつき かくりつ) → 1-1
ある事象 \(B\) が起きたと分かったときの事象 \(A\) の確率 \(P(A \mid B)\) です。 「\(B\) のもとでの \(A\) の確率」と読みます。ベイズの定理の土台となる、もっとも大事な道具です。 定義: \(P(A \mid B) = \dfrac{P(A \cap B)}{P(B)}\)
信頼区間(しんらい くかん) → 3-6
頻度論の区間推定で、「同じやり方で区間を作りつづければ、その95%が真の値を含む」という意味を持ちます。 一本の区間そのものに「95%の確率で真の値が入る」とは言えない点が、信用区間とのいちばんの違いです。
信用区間(しんよう くかん) → 3-5
ベイズの区間で、事後分布のうち確率が95%(など)集まる範囲です。 「真の値がこの区間に入る確率が95%」とそのまま言えるのが特徴。頻度論の信頼区間とは意味が違います。 比較: 3-6 信頼区間と信用区間
正規分布(せいき ぶんぷ) → 2-3
左右対称のつりがね型をした連続分布です。平均 \(\mu\) で中心が、標準偏差 \(\sigma\) で広がりが決まります。 自然界やデータに広く現れ、ベイズ推定でもよく登場します。 ベイズでの推定: 3-4 正規分布のベイズ推定

た行

単純ベイズ(たんじゅん ベイズ / ナイーブベイズ) → 5-6
「各特徴は互いに独立」と思いきって仮定して分類する、ベイズ的な機械学習の手法です。 仮定は乱暴でも実用ではよく当たり、迷惑メール判定などで使われます。 入口の例: 1-6 迷惑メールフィルタ
逐次更新(ちくじ こうしん) → 3-1
データを一つずつ取り込み、そのつど信念を更新していくやり方です。 「今回の事後分布が、次回の事前分布になる」のがポイントで、ベイズが学習する仕組みそのものです。
同時確率(どうじ かくりつ) → 1-2
2つの事象が両方とも起きる確率 \(P(A, B)\) です。分割表のマス目一つ分の確率にあたります。 これを片方の事象についてまとめ上げると周辺確率になります。
独立(どくりつ) → 1-2
一方が起きても、もう一方の確率が変わらない関係です。式では \(P(A \cap B) = P(A)\,P(B)\) が成り立ちます。 コインの1投目と2投目のような、互いに影響しない出来事の関係をいいます。

な行

二項分布(にこう ぶんぷ) → 2-2
成功確率が一定の試行を \(n\) 回くり返したとき、成功回数がしたがう離散分布です。 コインの表の回数や、アンケートの賛成人数などを表すのに使います。ベイズではベータ分布と組んで活躍します。
NUTS(ナッツ / No-U-Turn Sampler) → 4-5
HMC(ハミルトニアンモンテカルロ)を自動でうまく調整する改良版です。 どこまで進むかを賢く打ち切るので、細かな設定なしに高次元の事後分布を効率よく探索できます。PyMCの標準サンプラーです。

は行

ハミルトニアンモンテカルロ(HMC) → 4-5
事後分布をなめらかな地形に見立て、物理の運動のアナロジーでサンプルを進めるMCMCです。 でたらめに動くより遠くまで効率よく動けるため、高次元でも素早く探索できます。
頻度論(ひんどろん) → 3-6
確率を「同じ試行をくり返したときの長い目で見た割合」と考える立場です。 パラメータは固定された未知の定数とみなし、信頼区間や \(p\) 値を使います。ベイズと対比される考え方です。 比較: 3-6 頻度論 vs ベイズ
ベータ分布(ベータ ぶんぷ) → 2-6
0から1までの値をとる連続分布で、「確率そのものの確率」を表すのにぴったりです。 2つの形パラメータ \(\alpha, \beta\) で形が変わります。二項分布の共役事前分布として中心的な役割を果たします。 活用: 3-2 ベータ・二項モデル
ベイズの定理(ベイズ の ていり) → 1-3
事前確率と尤度から事後確率を求める公式 \( P(H \mid D) = \dfrac{P(D \mid H)\,P(H)}{P(D)} \) です。 「結果から原因をさかのぼる」というベイズ統計の心臓部にあたります。 入り口: 0-5 「逆向き」の確率
ベイズ更新(ベイズ こうしん) → 3-1
データを見て、事前分布を事後分布へと書き換える操作のことです。 ベイズの定理を分布全体に当てはめたもので、くり返せば逐次更新になります。
ベイズファクター(ベイズ ファクター) → 3-7
2つのモデルの周辺尤度の比で、「どちらのモデルがデータをうまく説明したか」を測るものさしです。 値が大きいほど、分子のモデルがデータに支持されたことを意味します。 材料: 周辺尤度 \(P(D)\)
変分推論(へんぶん すいろん) → 4-7
サンプリングを使わず、扱いやすい分布で事後分布を近づける最適化として推定する方法です。 MCMCより速い一方、近似の精度には注意が必要です。大規模なデータで重宝します。

ま行

メトロポリス・ヘイスティングス法(メトロポリス・ヘイスティングス ほう) → 4-3
もっとも基本的なMCMCです。次の候補をランダムに提案し、あるルールで受理か棄却かを決めながら歩きます。 正規化定数(周辺尤度)を計算しなくてよいのが大きな利点です。
モンテカルロ法(モンテカルロ ほう) → 4-2
たくさんの乱数を使って、面積や平均などの計算を近似する方法の総称です。 事後分布からサンプルを集めて、その平均や区間を読み取る——というベイズ計算の土台になります。
モデル比較(モデル ひかく) → 3-7
候補となる複数のモデルのうち、どれがデータをよく説明するかを比べることです。 ベイズではベイズファクターや周辺尤度を使って評価します。

や行

尤度(ゆうど) → 2-5
あるパラメータ(仮説)を仮定したときの、手元のデータの出やすさ \(P(D \mid H)\) です。 データを固定してパラメータの関数として見るのがポイント。「確率」と向きが逆である点に注意します。 部品としての役割: 1-4 事前確率・尤度・事後確率
尤度関数(ゆうど かんすう) → 2-5
尤度を、パラメータを変数とみなした関数として書いたものです。 この関数が高いパラメータほど、データをよく説明します。最尤推定はこの関数の山のてっぺんを探します。

ら行

連続型分布(れんぞくがた ぶんぷ) → 2-3
身長や時間のように、切れ目なく値をとる確率変数の分布です。 点の確率ではなく確率密度で表し、「確率 = 面積」として面積で確率を読み取ります。正規分布やベータ分布が代表例です。
ロジスティック回帰(ロジスティック かいき) → 5-3
結果が「はい/いいえ」のような2択のときに、その確率を予測する回帰です。 ベイズで扱うと、回帰係数を分布として推定でき、予測の不確かさまで語れます。
